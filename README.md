# webpack-starter

- create src, dist

```shell
npm init -y
npm i -D webpack webpack-cli
npm install uuid
npm remove uuid
```

- create webpack.config.js
-

```shell
# install loader
npm i -D sass style-loader css-loader sass-loader
```
